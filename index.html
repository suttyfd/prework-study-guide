CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • GitHub is a repository for code
  • GitBash is a command line linked with VS code
  • git status: checks what branch we are currently on
  • git checkout main - takes you into the "Main" branch
  • git pull origin main - puts you into the latest Main Branch"
  • git checkout -b ex:(feature/add-html): creates a new branch and switches to it
  • type in git status to confirm what branch you're in
  • git add -A We use the add -A command to commit all files in the working branch.
